Welcome to svgR

svgR is an R-package which permits the easy generation of SVG markup using the R programming language.

Some of the benefits are:

  • Resolution Independence (scales nicely.)
  • Easy Interactive Scripting
  • Dynamically generated
  • Extensive Animation capablities
  • Does not require saving image files to disk

It is a snap to use inside RStudio i Shiny Web Server and inside R documents, R-presentations, and slides.

Preview

One of the best sources to see what can be done is the svgR Users Guide, is contains several hundred short examples.

About

Motivation for the svgR architecture was inspired by an article by Hadley Wickham on Domain Specific Languages. Some additional structural variations are implemented to present a more R-like usage. These variations are optional and a strict SVG approach will still work. For details see StructuralEnhancements
